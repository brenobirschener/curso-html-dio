Conceitos iniciais sobre HTML

O que é HTML?

HTML, que significa HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem padrão para a criação e design de páginas web. Sua principal função é estruturar o conteúdo de uma página, indicando a função de cada elemento dentro do documento.

Confira as principais funcionalidades do HTML.

  2. Tag "p".
  3. Tag "mark".

  6. Tags "ol, ul, li".
  7. Tag "a".

  9. Tag "Blockquote".

Tags "h1 a h6".

As Heading Tags (H1, H2, H3, ...) são recursos de programação HTML utilizados para destacar títulos e subtítulos de uma página. Elas são utilizadas na hierarquização e estrutura de uma página.

Tag "p"

A tag "p" é utilizada para demarcar um parágrafo. O elemento HTML < p > (ou Elemento HTML Parágrafo) representa um parágrafo do texto. Parágrafos são, geralmente, representados em mídia visual , como bloco de texto que são separados dos blocos adjacente por espaços brancos verticais e/ou recuo de primeira-linha.

Tag "mark"

A tag HTML < mark > é usada para marcar ou destacar o texto de interesse ou relevância especial em um documento HTML. A tag < mark > é nova no HTML 5.

Tag "small".

O marcador "Small Element" ou fonte menor é utilizado para reduzir o tamanho da fonte num intervalo definido de um texto.

Tag "i, u, strong".

Os marcador i, u , strong servem para deixar o texto em italico, sublinhado e negrito respectivamente.

Tags "ol, ul, li".

As tags ol, ul e li serve para criar listar ordenadas, listas não ordenadas e os itens da lista respectivamente

Tag "a".

< a > O elemento < a > em HTML (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.

Tag "sub, sup".

O marcador < SUB > "Subscript Element" ou subscrito é utilizado escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo, a tag sup é o contrario.

Tag "Blockquote"

O Elemento HTML < blockquote > (ou Elemento HTML de citação de bloco) indica que o texto incluído é uma longa citação.
